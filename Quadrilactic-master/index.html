Quadrilactic

About:

Quadrilactic is a simple game about bouncing through space. Jump to climb higher and increase your score.

Built using typescript and HTML5 canvas. View on GitHub to find out more.

Controls:

  • Go left: A, Left
  • Go right: D, Right
  • Jump: W, Space, Up
  • Start Game: Enter, E
  • Change Volume: M, V
